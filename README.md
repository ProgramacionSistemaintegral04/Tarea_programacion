# TRABAJO DE PROGRAMACIÓN.
# UNAD - TERCER SEMESTRE - FASE 4.
Se desarrolló un sistema en Python orientado a objetos para gestionar clientes, servicios (reservas de salas, alquiler de equipos y asesorías) y reservas, sin usar base de datos. Implementa clases abstractas, herencia, polimorfismo, encapsulación y manejo robusto de excepciones con registro de logs. El programa simula más de 10 operaciones (válidas e inválidas), demostrando estabilidad ante errores. El código está documentado y alojado en un repositorio de GitHub.
